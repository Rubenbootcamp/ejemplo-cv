# 📄 Curriculum Vitae Interactivo

## 🎯 Descripción del Proyecto
Este proyecto consiste en un CV personal responsivo y profesional, desarrollado para demostrar habilidades en Front-end (HTML, CSS, JS) y el uso de frameworks como Bootstrap y librerías como jQuery.

## 🚀 Demo en Vivo
Puedes ver el proyecto desplegado aquí: **https://rubenbootcamp.github.io/ejemplo-cv/**

## ✨ Características Principales
* **Diseño Responsivo:** Construido con Bootstrap para una visualización óptima en móviles y escritorio.
* **Interactividad con JS/jQuery:** * Interruptor de tema (Claro/Oscuro).
    * Validación de formulario de contacto en tiempo real.
    * Efectos de animación al desplazarse por las secciones.
* **Código Limpio:** Separación estricta de responsabilidades (HTML, CSS, JS) y uso de etiquetas semánticas.

## 🛠️ Tecnologías Utilizadas
* HTML5 (Etiquetas semánticas)
* CSS3 (Estilos personalizados)
* Bootstrap 5 (Framework de diseño)
* JavaScript & jQuery (Interactividad)
* Git & GitHub (Control de versiones y despliegue)
